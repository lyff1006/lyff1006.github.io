docker安装mysql5.7

小羽 2020-07-20 dockermysql

# 1.拉取mysql5.7镜像

docker pull mysql:5.7.30
1

# 2.创建并运行mysql5.7容器

docker run -itd --name mysql-5.7 -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql:5.7.30
1

# 3.安装navicat可视化管理工具

链接：https://pan.baidu.com/s/1eBlYoEBbfFxrZqPxcWzkpA 提取码：wwnw

# 3.1 安装navicat

下载navicat，解压后，打开安装包，接着就是傻瓜式安装了

# 3.2 破解navicat

（1）打开破解工具，点击patch

（2）生成注册码，并粘贴到navicat中，点击激活，选择手动激活。

（3）将手动激活中生成的请求码复制到navicate中，点击激活码旁边的generate按钮，生成激活码，将激活码拷贝到navicate中完成激活
