“优雅”的摸鱼系列——豆瓣电影插件

小羽 2021-02-05 摸鱼电影chrome插件

老铁们，好久不见呀~

想不到吧，2021年某位靓仔的第一篇文章居然是摸鱼系列，哈哈哈。

先问大家一个问题，有没有遇到过那种在豆瓣中看到了某部想看的电影，但是电影上演的时间已经是几年前了。这时候我们也去不了电影院进行观看，到网络上找资源又很耗费时间。或者想直接白那个啥，嫖那个啥的童鞋。

咳咳咳，注意啦，你们想要的东西来了。

先说一下我们整个插件的运行流程

1.打开豆瓣的电影详情页面

2.通过chrome插件获取电影的名字

3.通过chrome插件像我们的后台服务发起请求

4.后台服务通过爬虫，爬取相关的电影信息并返回到chrome插件中

# 1.创建chrome插件项目

这里的话小羽直接使用自己搭建的一个脚手架插件进行项目的创建，该脚手架是基于vue3.x进行搭建的，详情可以点击这里去查看。

全局安装sulg-plugin-cli

npm install sulg-plugin-cli -g
1

查看sulg-plugin-cli版本，出现如下图所示，说明脚手架安装成功

sulgPlugin -v
1

创建项目

sulgPlugin create douban-movie-plugin
1

选择模板类型，default是不带任何ui框架，antd2.0则是基于ant-design-vue2.0框架的模板，elementPlus则为官方elementui的vue3版本。这里选择了elementPlus

然后选择模板的下载地址，可以选择github或者gitee，国内的小伙伴建议选择gitee，速度会更快一些 。

接着就是填入一些项目的相关信息，然后就会自动下载好我们的模板。
