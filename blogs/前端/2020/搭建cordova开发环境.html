搭建cordova开发环境

小羽 2020-08-08 cordovajavaandroid开发环境

cordova开发环境是基于android的，而android又是基于java的，因此搭建cordova开发环境需要先搭建android环境和java环境

# 1.搭建java环境

# 1.1 下载安装java

从官网上下载jdk，然后傻瓜式安装。

安装完成后，可以通过cmd命令（win+r键，输入cmd，回车），然后输入java -version测试Java是否安装成功。

# 1.2 配置环境变量

此电脑→右键→属性→高级系统设置→环境变量

# 1.2.1 配置JAVA_HOME

新建系统变量，变量名为JAVA_HOME，变量值为jdk的安装目录

# 1.2.2 配置Path

path变量中新增**%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;**，注意忒好通过编辑文本添加，否则win的编辑器会自动添加双引号，从而导致配置失败

# 1.2.3 配置CLASSPATH

新增系统变量，变量名为CLASSPATH，变量值为.;%Java_Home%\bin;%Java_Home%\lib\dt.jar;%Java_Home%\lib\tools.jar；

# 1.2.4 测试环境变量

通过cmd命令，分别输入java和javac

# 2.搭建android开发环境

# 2.1 下载安装android stdio

直接在android stdio的官网下载，然后傻瓜式安装即可

# 2.2 添加android环境

configure->setting

选择想要添加的android环境，我这边把android7.0以上的环境全部勾选上了

2.3 配置android环境变量

新增系统环境变量，变量名为ANDROID_HOME，变量值为sdk的安装路劲

path变量后追加%ANDROID_HOME%\build-tools;%ANDROID_HOME%\platform-tools;%ANDROID_HOME%\tools;

# 2.3 测试android环境

cmd命令中输入adb

# 3.搭建gradle环境

gradle官网：https://gradle.org/

下载gradle：https://gradle.org/releases/

下载完，解压。然后配置环境变量。

1.新增GRADLE_HOME系统环境变量，变量值为gradle的路径

2.path系统环境变量后面追加**%GRADLE_HOME%\bin**;

# 4.安装cordova

前置条件：node.js环境

4.1 全局安装cordova

npm install cordova -g
1

4.2 添加platform

cordova platform add android
1

4.3 创建cordova项目

cordova create test
1

4.4 查看cordova 环境配置

cordova requirements
1

4.5 打包cordova项目（android）

cordova build android
1
